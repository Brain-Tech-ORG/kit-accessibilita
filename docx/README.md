# Template Word accessibili

Cinque template `.docx` conformi a **WCAG 2.1 AA**:

| File | Quando usarlo |
|------|---------------|
| `lettera-istituzionale.docx` | Comunicazioni formali: enti pubblici, organizzazioni |
| `relazione.docx` | Relazione/report con frontespizio, sommario, capitoli numerati |
| `verbale-riunione.docx` | Verbale di riunione (presenze, ODG, decisioni, azioni) |
| `modulo-richiesta.docx` | Istanze e richieste verso enti, con sezione privacy |
| `dispensa-didattica.docx` | Materiale formativo con obiettivi, glossario, autovalutazione |

## Caratteristiche di accessibilità incorporate

- Lingua impostata su `it-IT`
- Font Arial 11pt (sans-serif ad alta leggibilità)
- Stili "Titolo 1/2/3" già configurati con colori a contrasto sufficiente
- Tabelle con prima riga dichiarata come riga di intestazione
- Interlinea 1.15, margini 2.5 cm
- Proprietà del documento (titolo, autore, lingua) compilate

## Come usarli

1. Apri il `.docx` desiderato.
2. *File → Salva con nome*: salva una copia con un nome significativo.
3. **Sostituisci** i segnaposto `[…]` con i tuoi contenuti.
4. **Per i titoli**: usa lo *stile* "Titolo 1/2/3" dalla galleria stili. Non ingrandire il testo manualmente.
5. **Per le immagini**: tasto destro → *Modifica testo alternativo* e descrivi cosa rappresentano.
6. **Compila i metadati**: *File → Informazioni → Proprietà*.

## Esporta in PDF accessibile

*File → Salva con nome → PDF → Opzioni* → spunta:

- *Tag della struttura del documento per l'accessibilità*
- *Crea segnalibri usando: Intestazioni*

## Verifica

*Revisione → Verifica accessibilità*: deve restituire **0 errori** e **0 avvisi**.
